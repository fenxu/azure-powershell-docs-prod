# Remove-AzureADApplicationOwner

## SYNOPSIS
Removes an owner from an application.

## SYNTAX

```
Remove-AzureADApplicationOwner -ObjectId <String> -OwnerId <String>
```

## DESCRIPTION

## EXAMPLES

### Removes an owner from an application
```
Remove-AzureADApplicationOwner -ObjectId 3ddd22e7-a150-4bb3-b100-e410dea1cb84 -OwnerId c13dd34a-492b-4561-b171-40fcce2916c5
```
