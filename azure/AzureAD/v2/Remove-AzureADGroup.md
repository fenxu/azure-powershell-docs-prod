# Remove-AzureADGroup

## SYNOPSIS
Delete a group by objectId.

## SYNTAX

```
Remove-AzureADGroup -ObjectId <String>
```

## DESCRIPTION

## EXAMPLES

### Remove a group
```
Remove-AzureADGroup -ObjectId 11fa5e1e-737c-40c5-835e-416ae3959606
```

## PARAMETERS

### -ObjectId
The unique identifier of a group in Azure Active Directory (ObjectId)
