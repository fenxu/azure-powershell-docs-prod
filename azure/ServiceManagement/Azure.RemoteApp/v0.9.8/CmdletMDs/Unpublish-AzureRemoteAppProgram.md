# Unpublish-AzureRemoteAppProgram

## SYNOPSIS
Unpublishes a RemoteApp program.

## SYNTAX

```
Unpublish-AzureRemoteAppProgram [-CollectionName] <String> [[-Alias] <String[]>] [-Profile <AzureProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Unpublish-AzureRemoteAppProgram** cmdlet unpublishes an Azure RemoteApp program.
After you unpublish a program, it is no longer available to the users of a RemoteApp collection.
