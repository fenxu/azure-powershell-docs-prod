# Get-AzureADTenantDetail

## SYNOPSIS
Retrieves the details of a tenant in Azure Active Directory

## SYNTAX

```
Get-AzureADTenantDetail [-Top <Int32>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```
